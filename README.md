# Horai
[![License: CC BY-NC 4.0](https://licensebuttons.net/l/by-nc/4.0/80x15.png)](https://creativecommons.org/licenses/by-nc/4.0/)

Horai is a work-in-progress planner and note-taking app, designed to be cross-platform and shared across devices, based upon [Notes](https://github.com/starsflower/notes) and other planning apps.

## Setup
Horai is a Python powered Flask API that (will) provides front-end pages on demand. It can run with any database supported by SQLAlchemy.
